# AI Agent Sync Workflow

Sync and install the latest skills from the agentools repository.

## Steps

1. Update cache from GitHub and reinstall all skills to platforms:
```bash
agentools install
```

## Notes

- Requires `agentools` npm package to be installed globally
- If not installed, run: `npm install -g agentools`
- Skills source: https://github.com/dongitran/agentools
